# Bill of Materials (BOM)

## Рекомендуемый контроллер узла

**Primary:** Raspberry Pi Pico 2 W / Pico 2 WH.

- **Pico 2 WH**: функционально тот же Pico 2 W, но с уже припаянными header pins — лучший выбор для быстрого prototyping.
- **Pico 2 W**: функционально тот же контроллер без припаянных header pins — удобнее для компактной интеграции и кастомного монтажа.

## Минимальный стартовый набор (MVP)

| Компонент | Кол-во | Обяз. | Назначение |
|---|---:|:---:|---|
| Raspberry Pi Pico 2 W или Pico 2 WH | 1 | Да | primary node controller |
| MFRC522 RFID reader module | 1 | Да | детекция поезда |
| RFID tags/cards (13.56 MHz) | 3-5 | Да | идентификация поездов |
| Servo SG90/MG90S | 2 | Да | переключение 2 стрелок на старте |
| Внешний БП 5V 3A (минимум) | 1 | Да | питание servo |
| USB data cable (Pico ↔ Raspberry Pi) | 1 | Да | MVP transport (USB Serial) |
| Breadboard + Dupont wires | 1 набор | Да | макетное подключение |
| Конденсатор 470-1000µF, 6.3V+ | 1-2 | Да | сглаживание пиков servo |
| Raspberry Pi 4/5 или mini PC | 1 | Да | mosquitto + backend + UI + serial bridge |
| microSD (для Pi) | 1 | Да | ОС/данные |

| Шлагбаум actuator | по плану | Опц. | будущая автоматика |

## Где не экономить
- Питание 5V с запасом по току для servo.
- Качество проводов и контактов.
- Отдельный контур питания для servo при 2+ приводах.
- USB кабель с надёжной передачей данных (не только зарядка).

## Что важно про уровни напряжения
- Pico GPIO = **3.3V logic only**.
- MFRC522 лучше сочетается с Pico по логическим уровням.
- 5V сигналы нельзя подавать на GPIO Pico напрямую.
