# Bill of Materials (BOM)

## Минимальный стартовый набор (MVP)

| Компонент | Кол-во | Обяз. | Назначение |
|---|---:|:---:|---|
| Arduino Nano/Uno совместимая плата | 1 | Да | node controller |
| MFRC522 RFID reader module | 1 | Да | детекция поезда |
| RFID tags/cards (13.56 MHz) | 3-5 | Да | идентификация поездов |
| Servo SG90/MG90S | 1-2 | Да | переключение стрелки |
| Внешний БП 5V 3A (минимум) | 1 | Да | питание servo и/или node |
| Breadboard + Dupont wires | 1 набор | Да | макетное подключение |
| Конденсатор 470-1000µF, 6.3V+ | 1-2 | Да | сглаживание пиков servo |
| Raspberry Pi 4/5 или mini PC | 1 | Да | mosquitto + backend + UI |
| microSD (для Pi) | 1 | Да | ОС/данные |

## Для расширения

| Компонент | Кол-во | Опц. | Назначение |
|---|---:|:---:|---|
| Доп. MFRC522 | +1 на точку | Да | новые точки считывания |
| Доп. servo | +1 на стрелку | Да | масштабирование |
| PCA9685 servo driver | 1 | Опц. | управление множеством servo |
| Оптопары/драйверы | по схеме | Опц. | развязка и надёжность |
| DC-DC buck converter | 1+ | Опц. | распределение питания |
| Светофоры/LED модули | по плану | Опц. | будущие сигналы |
| Шлагбаум actuator | по плану | Опц. | будущая автоматика |

## Где не экономить
- Питание 5V с запасом по току.
- Качество проводов и контактов.
- Отдельный контур питания для servo при 2+ приводах.
