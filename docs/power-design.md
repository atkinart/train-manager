# Power Design

## Ключевой принцип

**Логика (Arduino/RFID) и силовая часть (servo) должны быть разделены по питанию, но иметь общую землю.**

## Рекомендованная схема для MVP

- Arduino: USB 5V или стабильный 5V.
- MFRC522: от 3.3V Arduino.
- Servo: от отдельного 5V БП (минимум 3A для старта с запасом).
- Общая земля: соединить GND Arduino и GND servo PSU.

## Почему нельзя питать несколько servo от Arduino 5V

- Пусковой ток servo может быть кратно выше номинала.
- Падение напряжения вызывает reset контроллера.
- Возникают случайные ошибки чтения RFID и ложные команды.

## Raspberry Pi питание

- Использовать штатный БП (обычно 5V 3A+).
- Не питать Pi от той же линии, где дёргаются servo, без фильтрации/развязки.

## Ток и запас

Грубая оценка:
- SG90: до ~500-700mA на пиках.
- 2 servo одновременно: до ~1.4A пиково.
- Закладывать минимум x2 запас по току от расчётного среднего.

## Будущая силовая шина

Для 4+ servo:
- отдельная 5V power bus,
- предохранители по веткам,
- распределительная плата,
- при необходимости PCA9685 + внешнее питание.
