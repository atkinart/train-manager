# Power design

## Базовая идея

Логика и силовая часть разделяются:
- **Logic domain (3.3V)**: Pico 2 W + MFRC522.
- **Power domain (5V)**: servo motors.
- Обе части обязаны иметь общий reference: **COMMON GND**.

## Правила питания

- Pico GPIO: только 3.3V уровень.
- MFRC522: питание 3.3V.
- Servo: внешний 5V PSU (не от Pico).
- Общая земля между Pico, MFRC522 и servo PSU.

## Важные запреты

- Нельзя подавать 5V сигналы на GPIO Pico.
- Нельзя питать servo от Pico 3.3V/VSYS.
- Нельзя подключать 5V UART напрямую в Pico RX без согласования уровней.
- Нельзя отлаживать всё сразу; делайте по этапам.

## Минимальная последовательность проверки

1. Pico + MFRC522 без servo.
2. Проверить чтение RFID.
3. Добавить servo signals (без движения под нагрузкой).
4. Подать отдельное 5V питание servo.
5. Проверить отсутствие reset/шума при движении.
