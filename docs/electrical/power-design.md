# Electrical Power Design

## Разделение питания

- Логика: Pico + MFRC522 = 3.3V.
- Силовая часть: servo = отдельные 5V.
- Общий reference: common ground.

## Почему это важно

Servo создают пиковые токи, из-за которых:
- контроллер может перезагружаться;
- RFID может читать нестабильно;
- команды могут теряться.

## Минимальный практический набор

- Внешний 5V БП на 3A+.
- Конденсатор 470-1000µF рядом с servo питанием.
- Короткие провода GND.
