# MVP пошаговое безопасное подключение

Эта инструкция специально сделана для первого запуска без риска «спалить» модуль или запутаться в проводах.

1. **Сначала не подключайте servo.**
2. Подключите только Arduino и один MFRC522.
3. Проверьте, что на MFRC522 подан именно `POWER 3.3V`.
4. Подключите `COMMON GND` между Arduino и MFRC522.
5. Подключите `SPI BUS` (SCK/MOSI/MISO/CS) и RST.
6. Запустите тест: убедитесь, что reader читается.
7. Только после этого подключите `servo signal` к Arduino.
8. Только после этого подключите внешний `POWER 5V` для servo.
9. Проверьте `COMMON GND` между Arduino и servo PSU.
10. Только потом подключайте Raspberry Pi через `USB SERIAL`.
11. Проверьте обмен данными с Raspberry Pi.
12. Лишь после успешных тестов подключайте второй reader или расширяйте узел.

## Что проверять после каждого шага

- Нет ли нагрева модулей.
- Нет ли запаха перегрева.
- Нет ли самопроизвольных reset Arduino.
- Стабильна ли работа RFID несколько минут подряд.
- Двигаются ли servo без рывков и пропаданий управления.
