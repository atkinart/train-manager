# PCA9685 Future Expansion

## Когда нужен PCA9685

Когда количество servo растёт (обычно 4+), лучше вынести генерацию PWM в отдельный драйвер.

## Что это даст

- Стабильные PWM каналы для большого числа приводов.
- Меньше нагрузки на Pico.
- Удобное масштабирование по I2C.

## Важно

- PCA9685 не отменяет внешнее питание servo.
- Common ground остаётся обязательным.
- Для MVP с 2 servo PCA9685 не обязателен.
