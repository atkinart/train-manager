# MFRC522 Connection (Pico 2 W)

## Важное правило

MFRC522 подключается к Pico как 3.3V устройство (логика и питание).

## Рекомендуемые соединения

- SDA/SS -> GP5
- SCK -> GP2
- MOSI -> GP3
- MISO -> GP4
- RST -> GP6
- 3.3V -> 3V3(OUT)
- GND -> GND

## Практические советы

- Делайте SPI провода как можно короче.
- Избегайте «рыхлых» breadboard контактов.
- Если есть нестабильность чтения — сначала проверяйте питание и GND, потом код.
