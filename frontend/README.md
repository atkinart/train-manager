# Frontend (Vite + React)

Почему React:
- простой старт для UI-панели;
- легко добавить интерактивную схему и состояние в реальном времени;
- понятная структура компонентов для дальнейшего роста.

## Run

```bash
cd frontend
npm install
npm run dev
```

Env:
- `VITE_API_BASE_URL` (по умолчанию пусто, ожидается reverse proxy или same host).
